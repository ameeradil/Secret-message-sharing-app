# 🔒 Secret Message Sharing App

> Securely share secret messages via custom links or encrypted text.  
> A simple app that lets users enter a secret and generate a sharable link that reveals the message to the recipient.

---

## ✨ Features

✅ Accept a secret message from the user  
🔗 Generate a unique sharable link or token  
🛡️ Optionally encode/encrypt the message  
📩 Reveal the secret only when the link is accessed    
🐍 Optional Python backend for message storage or encryption handling

---

## 🧠 How It Works

1. You write a secret message in the input field  
2. The app encodes/encrypts the message in a shareable format  
3. A link or token is generated for the secret  
4. Anyone with the link or token can view the secret message once

This is a common pattern used in secret sharing apps — encoded secrets in links, burn-after-reading style sharing, or client-side transformation. :contentReference[oaicite:1]{index=1}
